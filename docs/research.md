### 1.2 核心理念
引用荣格的话作为产品哲学基础：
> "Unless you make the unconscious conscious, it will direct your life and you will call it fate."
> 
> "除非你让潜意识变得有意识，否则它将主导你的生活，而你将称之为命运。"

### 1.3 三大核心原则
1. **AI 智能，科学分析** - 融合现代 AI 技术与传统占卜智慧，通过大数据分析和机器学习提供客观解读，去神秘化
2. **文化传承，理性态度** - 以开放理性的态度传承东方智慧，去糟粕存精华，避免迷信和神秘化
3. **自我探索，独立思考** - 相信每个人都是自己命运的解读者，通过 AI 赋能让用户自主分析、独立思考

## 二、产品功能矩阵

### 2.1 八字算命体系 (核心功能)

| 功能 | 描述 | 输入要素 |
|------|------|----------|
| **八字计算** | AI 智能八字分析系统，揭示命盘特征和发展模式 | 姓名、性别、出生日期（阳历/农历）、出生时间、出生地点 |
| **每日运势** | 基于八字的日运分析，帮助把握每日运势和机会 | 已保存的八字信息 |
| **八字合婚** | 两人八字配对分析，揭示伴侣间的关系兼容性 | 双方八字信息 |
| **事业合盘** | 商业伙伴关系分析，了解合作潜力和挑战 | 双方八字信息 |
| **婆媳合盘** | 婆媳关系分析，揭示潜在互动和改善方法 | 双方八字信息 |
| **闺蜜合盘** | 朋友关系分析，揭示兼容性和友谊潜力 | 双方八字信息 |
| **父子关系** | 父子关系分析 | 双方八字信息 |
| **母子关系** | 母子关系分析 | 双方八字信息 |
| **朋友关系** | 普通朋友关系分析 | 双方八字信息 |
| **领导下属** | 职场上下级关系分析 | 双方八字信息 |

#### 八字计算的输入设计亮点
- **双历法支持**: 阳历 / 农历 切换
- **双输入模式**: 日期输入 / 直接输入八字
- **时间精度**: 精确到分钟，并计算真太阳时
- **地点定位**: 搜索选择出生地，获取精确经纬度
- **早晚子时**: 可选开关，处理子时换日问题
- **不知道出生时间**: 可选择关闭，仍可获得年月日三柱分析

#### 八字输出内容
- 四柱命盘（年柱、月柱、日柱、时柱）
- 五行分析与平衡
- 性格洞察
- 事业方向
- 财运特征
- 健康运势
- 大运流年

### 2.2 紫微斗数

| 功能 | 描述 |
|------|------|
| **紫微排盘** | 根据出生时间生成十二宫命盘，分析主星分布 |

#### 特点
- 精准排盘：根据出生时辰计算十二宫位和主星分布
- 十二宫位分析：命宫、事业、财帛、夫妻等
- 星曜与四化：14主星、六吉星、六煞星、四化分析
- **必须提供准确出生时辰**

### 2.3 周易占卜

#### 梅花易数 · 每日决策
| 起卦方式 | 描述 |
|----------|------|
| 时间起卦 | 基于当前时间自动起卦 |
| 数字起卦 | 用户输入数字起卦 |
| 随机数起卦 | 系统生成随机数起卦 |
| 手动起卦 | 用户自定义卦象 |

**特色功能**:
- 多角度观察（不只是是/否，提供多层次观点）
- 个性化参考
- 清晰表达
- 模式观察

#### 六爻占卜
| 起卦方式 | 描述 |
|----------|------|
| 手动起卦 | 点击爻位设置阴阳，选择变爻位置 |
| 时间起卦 | 基于当前时间 |
| 数字起卦 | 用户输入数字 |
| 随机数起卦 | 系统随机生成 |

**核心要素**:
- 六爻卦象预览（初爻到上爻）
- 上卦/下卦显示
- 变爻标识（少阳/少阴/老阳/老阴）
- 占卜日期时间记录
- 纳甲体系分析
- 世应关系
- 六亲配置
- 六神状态

### 2.4 塔罗牌

**输入方式**:
- 用户输入问题（文本框）
- 预设问题模板（快速选择）

**预设问题示例**:
- 创业还是留在大公司更适合我？
- 对方不喜欢我，我该如何让自己放下？
- 为何我在社交场合难以做真实的自己？
- 异地恋还要继续坚持吗？
- 如何处理与同事的项目分歧？
- 如何与伴侣谈论金钱问题？
- 如何应对突如其来的职位调整？

**解读风格（亮点！）**:
| 风格 | 描述 |
|------|------|
| **标准** | 专业客观的塔罗牌解读，提供平衡全面的命运分析 |
| **火神** | 直白犀利的解读，毫不留情指出问题，但给予实用建议 |
| **月光** | 温柔优雅的解读风格，着重心灵疗愈与精神指引 |
| **维斯顿** | 如教授般理性分析，结合哲学与心理学提供深度见解 |

**输出形式**:
- 抽取三张塔罗牌
- AI 精准解读
- 即时反馈

### 2.5 实用工具

#### AI 老黄历
- 传统中国黄历查询
- 每日宜忌分析
- 活动择日参考

#### 历史记录
- 保存用户的分析历史
- 方便回顾和对比

## 三、技术实现细节

### 3.1 时间计算
- **真太阳时计算**: 根据出生地经纬度，将标准时间转换为真太阳时
- **早晚子时处理**: 可选开关处理 23:00-01:00 的换日问题
- **农历转换**: 阳历/农历双向转换

### 3.2 地理定位
- 搜索式地点选择器
- 自动获取经纬度
- 用于计算真太阳时

### 3.3 起卦算法
- **时间起卦**: 根据年月日时数字计算
- **数字起卦**: 用户输入数字转换为卦象
- **随机数起卦**: 使用随机数生成器
- **手动起卦**: 用户直接设置每一爻

### 3.4 AI 分析引擎
- 结合传统命理理论与大数据分析
- 个性化解读生成
- 多维度分析框架

## 八、产品亮点总结

### 8.1 做得好的地方

1. **场景化产品设计**
   - 不只是"算命"，而是切入具体生活场景
   - 婆媳关系、闺蜜配对、领导下属等细分场景
   - 塔罗牌的预设问题模板

2. **科学包装策略**
   - 强调 AI 智能、大数据分析、客观解读
   - 去神秘化，理性化
   - 引用荣格心理学背书

3. **多风格解读**
   - 塔罗牌的四种解读风格是很好的创新
   - 满足不同用户的心理需求

4. **完整的知识体系**
   - 大量理论内容建立专业感
   - 双重作用：SEO + 用户教育

5. **低门槛策略**
   - 免费使用基础功能
   - 无需注册即可体验
   - 清晰的付费价值点

6. **多语言国际化**
   - 面向全球华人市场
   - 日文支持拓展亚洲市场

7. **技术细节专业**
   - 真太阳时计算
   - 早晚子时处理
   - 精确的地理定位

### 8.2 可改进/差异化空间

1. **交互形式**
   - 目前是填表出报告的模式
   - 缺少"命理师对谈"的沉浸感
   - **机会**: 做对话式交互

2. **用户关系**
   - 缺少持续陪伴和跟踪
   - **机会**: 做"你的专属命理师"

3. **社交属性**
   - 目前是工具化产品
   - **机会**: 加入分享、对比、社区元素

4. **本土化深度**
   - 偏国际化设计
   - **机会**: 更中国风、更接地气

5. **触达渠道**
   - 仅有网页端
   - **机会**: 微信生态、小程序、公众号

## 九、对"赛博命理师"项目的启示

### 9.1 可借鉴

1. **AI + 玄学的定位** - 用科技重新包装传统文化
2. **场景化产品设计** - 从用户问题出发，而非功能出发
3. **免费 + 付费漏斗** - 降低门槛，价值付费
4. **理论知识沉淀** - 建立专业权威感
5. **免责声明** - 法律合规必须做

### 9.2 可差异化

1. **对话式交互**
   - 不是填表出报告
   - 而是和"命理师"对话
   - 更有温度，更像咨询

2. **持续陪伴**
   - 不是一次性服务
   - 而是"你的命理师随时在"
   - 结合我们的 Agent 能力

3. **个性化人设**
   - FateMaster 的塔罗风格是好开始
   - 我们可以有完整的"命理师人设"
   - 有性格、有口头禅、有温度

4. **微信生态**
   - 公众号、小程序、企微
   - 符合中国用户习惯

5. **社交裂变**
   - 合盘分享
   - 运势 PK
   - 闺蜜群组
