  - 性别（必填，男/女）

- [ ] 提交 loading 状态

- [ ] 命盘卡片：四柱（年/月/日/时）+ 天干地支 + 五行

- [ ] 日主分析：日主类型 + 特征描述

- [ ] 底部免责声明

- [ ] localStorage 记住用户信息
- [ ] 二次访问自动读取
- [ ] "重新输入"按钮
- [ ] 结果展示：
  - 日期信息（阳历 + 农历 + 干支）
  - 整体运势（1-5 星）
  - 分项运势（事业/财运/感情/健康）

**其他页面**

### 7.2 用户体验建议

1. **表单体验**
   - 出生日期默认值设为合理年份（如 1990）
   - 时辰选择带时间范围提示（如"子时 23:00-01:00"）
   - 错误提示要具体明确

2. **结果页体验**
   - AI 解读生成时间可能较长，需明确的加载提示
   - 考虑"打字机效果"逐字显示，提升等待体验
   - 结果页提供"重新测算"入口

3. **转化引导**
   - 结果页底部可加"查看每日运势"引导
   - 首页 CTA 更明显

## 八、美术虾意见 🎨

### 8.1 设计规范落地检查

**配色 (DESIGN_SPEC 第二章)**
- [ ] 背景渐变是否正确：`#1a1a2e → #16213e → #0f0f1a`
- [ ] 金色强调色是否统一：`#ffd700`
- [ ] 五行配色是否应用：
  - 金 `#ffd700` / 木 `#4ade80` / 水 `#38bdf8` / 火 `#f97316` / 土 `#a78b5a`

**字体 (DESIGN_SPEC 第三章)**
- [ ] 标题是否使用思源宋体 (Noto Serif SC)
- [ ] 正文是否使用系统无衬线字体
- [ ] 字体大小层级是否清晰

**组件样式 (DESIGN_SPEC 第五章)**
- [ ] 主按钮：金色渐变背景 + 深色文字 + hover 上浮发光
- [ ] 次按钮：透明背景 + 金色边框
- [ ] 卡片：毛玻璃效果 + 金色微光边框 + hover 上浮
- [ ] 输入框：深色背景 + focus 金色边框 + 上方标签
- [ ] 星级评分：金色填充星 + 灰色空星

### 8.2 视觉效果建议

1. **首页 Hero**
   - 建议加背景装饰（星空粒子或八卦图水印）
   - 标题"赛博命理师"可加金色发光效果
   - 副标题与 CTA 之间间距调整

2. **功能卡片**
   - "即将上线"卡片建议加灰色遮罩 + 降低透明度
   - 图标大小统一（建议 48px）
   - hover 动效要有发光效果

3. **八字结果页**
   - 命盘四柱卡片要突出，建议加金色边框
   - 五行分析条建议用对应颜色填充
   - AI 解读区块之间加分隔线

4. **加载动画**
   - 建议使用八卦旋转动画（DESIGN_SPEC 5.8 节）
   - 骨架屏样式要符合深色主题

5. **移动端**
   - 导航栏收起为汉堡菜单
   - 卡片网格改为单列
   - 命盘四柱改为 2x2 布局
