## 一、产品概述

### 1.1 产品定位
赛博命理师是一款 **AI 驱动的东方命理分析网站**，融合传统命理学与现代人工智能技术，为用户提供科学、理性、有参考价值的命理分析服务。

### 1.2 产品愿景
让每个人都能便捷地了解自己的命理特征，在人生重要决策时获得有价值的参考。

### 1.3 核心价值
- **AI 智能解读**: 利用大语言模型生成个性化、易理解的命理分析
- **科学理性**: 去神秘化，以理性态度传承东方智慧
- **免费易用**: 基础功能免费，无需注册即可使用

### 2.1 用户画像
| 维度 | 描述 |

| 职业 | 职场人群、自由职业者、创业者 |
| 特征 | 对传统文化有兴趣但不迷信，理性思考 |

## 六、UI 设计规范

> 📄 **完整设计规范**: [DESIGN_SPEC.md](./DESIGN_SPEC.md)

### 6.1 设计理念

**「赛博东方」— 科技感与东方美学的融合**

- **现代科技感**: 深色界面、发光效果、渐变色彩
- **东方神秘感**: 传统纹样、阴阳五行、书法元素
- **专业可信赖**: 简洁克制、层次分明、信息清晰

**设计原则**:
1. 深色优先 — 减少视觉疲劳，营造神秘氛围
2. 金色点睛 — 金色作为强调色，体现东方贵气
3. 留白呼吸 — 适度留白，让信息有节奏感
4. 动静结合 — 微妙动效提升体验，但不喧宾夺主

### 6.2 配色方案

**主色板**
| 色彩角色 | 色值 | 用途 |
|----------|------|------|
| 深夜蓝 | `#1a1a2e` | 主背景、容器底色 |
| 星空紫 | `#16213e` | 次级背景、卡片底色 |
| 深底色 | `#0f0f1a` | 页面最底层背景 |

**强调色**
| 色彩角色 | 色值 | 用途 |
|----------|------|------|
| 皇金 | `#ffd700` | 主按钮、重要标题、图标 |
| 暗金 | `#b8860b` | 次要强调、边框 |
| 金色渐变 | `#ffd700 → #f4a460` | 按钮渐变、特殊强调 |

**五行配色**
| 五行 | 色值 | 说明 |
|------|------|------|
| 金 | `#ffd700` | 金色 |
| 木 | `#4ade80` | 翠绿 |
| 水 | `#38bdf8` | 天蓝 |
| 火 | `#f97316` | 橙红 |
| 土 | `#a78b5a` | 棕黄 |

**文字色**
| 角色 | 色值 | 用途 |
|------|------|------|
| 主文字 | `#f5f5f5` | 正文、标题 |
| 次文字 | `#a0a0a0` | 次要信息、描述 |
| 弱文字 | `#666666` | 占位符、禁用态 |
| 高亮 | `#ffd700` | 强调文字 |

### 6.3 字体规范

**字体家族**
- **标题**: Noto Serif SC (思源宋体) — 体现东方韵味
- **正文**: PingFang SC / system-ui — 保证可读性
- **等宽**: SF Mono / Fira Code — 八字显示

**字体大小**
| 级别 | 桌面端 | 移动端 | 用途 |
|------|--------|--------|------|
| H1 | 48px | 32px | 页面主标题 |
| H2 | 36px | 24px | 区块标题 |
| H3 | 24px | 20px | 卡片标题 |
| Body | 16px | 14px | 正文 |
| Small | 14px | 12px | 辅助文字 |

### 6.4 响应式断点

| 断点 | 宽度 | 布局 | 列数 |
|------|------|------|------|
| 手机 | <640px | 单栏 | 1 |
| 平板 | 768-1024px | 双栏 | 2 |
| 桌面 | >1024px | 多栏 | 3-4 |

### 6.5 组件规范

**按钮**
| 类型 | 样式 |
|------|------|
| 主按钮 | 金色渐变背景，深色文字，hover 上浮 + 发光 |
| 次按钮 | 透明背景，金色边框，hover 背景微亮 |
| 幽灵按钮 | 透明背景，灰色边框，hover 边框变亮 |
| 禁用态 | 灰色背景，不可点击 |

**卡片**
- 背景: `rgba(22, 33, 62, 0.8)` + 毛玻璃效果
- 边框: `rgba(255, 215, 0, 0.1)` 金色微光
- 圆角: 16px
- 悬停: 上浮 4px + 边框变亮 + 金色光晕

**输入框**
- 背景: `rgba(15, 15, 26, 0.8)`
- 边框: `rgba(255, 255, 255, 0.1)`
- 聚焦: 金色边框 + 金色光晕
- 标签: 上方标签式
- 错误: 红色边框 + 错误提示

**星级评分** (运势展示)
- 填充星: 金色 `#ffd700`
- 空星: 灰色 `#4a4a5a`

**五行进度条**
- 各五行使用专属颜色
- 动画过渡 0.5s

### 6.6 动效规范

| 场景 | 时长 | 缓动函数 |
|------|------|----------|
| 悬停状态 | 200ms | ease |
| 展开/收起 | 300ms | ease-out |
| 页面切换 | 400ms | ease-in-out |
| 加载动画 | 1500ms | linear |

**动效原则**:
- 克制使用，动效是点缀不是主角
- 尊重用户的 `prefers-reduced-motion` 设置
